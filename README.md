> A Nutrition tracker built in with MongoDb, Express, React, Node, GraphQL and the Food Center API.

## Prerequisites

- node 14.15.5
- npm 6.14.11

## Install

```sh
npm install
```

```sh
cd client && npm install
```

## Usage

### Server

```sh
npm run build:server
```

```sh
npm run start
```

### Client

```sh
cd client && npm run start
```
